Cómo funciona

Todo ocurre en el servidor, en este orden, cada vez que pulsas «Generar».

01

La imagen llega

Se genera con Pollinations.ai o la subes tú desde tu dispositivo.

02

Grok la describe

El modelo de visión de xAI (Grok) escribe una descripción objetiva en el nivel de lectura elegido, y la traduce si lo pides.

03

gTTS crea el audio

Se genera un mp3 real por idioma, listo para reproducir aquí mismo o para enviar por WhatsApp, correo o donde lo necesite la persona.

Poner esta página en un servidor

Es una app Flask normal. Resumen rápido — instrucciones completas en el README.md del proyecto.

En local

  1. pip install -r requirements.txt
  2. Copia .env.example a .env y añade tu GROK_API_KEY
  3. python app.py y abre http://localhost:5000

En un servidor (producción)

  1. Sube la carpeta del proyecto
  2. Define la variable de entorno GROK_API_KEY en el panel del hosting
  3. Arranca con gunicorn app:app

Voz Visible — construida con Flask, Grok (xAI) para descripción y traducción, gTTS para voz y Pollinations.ai para imagen. Los archivos generados se guardan en static/generated/ en el servidor.
